PlatformSettings

User platform settings information.

Overview

It is a utility that provides information about the user's platform that is relevant for customizing the application Window. This is information about the theme, operating system, order of the window control buttons, border shape and things like that.

Class signature

PlatformSettings(platform_integration: bool = True)

Parameters

  • platform_integration: Type bool. Default is True

    Using False will not bring platform information, it will bring library defaults. Using True will obtain platform information, only when possible and if it is supported.
